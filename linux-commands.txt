# Linux Commands in Summary

1. pwd: shows the current location in the file system that you are in.

2. ls: displays contents of current directory.

3. cd: used to move to different locations in the file system.

4. man: shows man page for given command.

5. touch: creates an empty file.

6. echo: redirects input to STDOUT.

7. >: overwrites an already existing file or a new file is created provided the mentioned file name isn't there in the directory.

8. >>: appends an already present file.

9. cat: displays contents of file(s).

10. rm: removes files.

11. mv: moves files from origin to destination.

12. mkdir: creates a directory.

13. cp: copies files/directory from one location to another.

14. rmdir: removes a directory.
